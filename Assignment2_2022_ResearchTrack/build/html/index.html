Action_client_A Module

This node implements an action client node that allows the user to set the target (x,y) or to cancel it by keyboard input in the console; then, it publish the robot position and velocity as a custom message (pos_vel), by relying on the values publish on the topic /odom.

Subscriber:

/odom

Publish:

/pos_vel

Action client:

/reaching_goal

Callback function to get the position and velocity and publish them as a custom message (pos_vel).

Parameters:

msg (Pos_vel) – the position and velocity of the robot

scripts.action_client_A.main()[source]

This function initializes the ROS node, the publisher and the subscriber, and calls the client function.

This function implements the action client node that allows the user to set the target (x,y) or to cancel it.

Service_node_B Module

This node implements a service node that, when it is called, prints the number of goals reached and cancelled;

Service:

/goal_n

Subscriber:

/reaching_goal/result

scripts.service_node_B.goal_number(msg)[source]

This callback function subscribes to the ‘/reaching_goal/result’ topic and updates the number of reached and cancelled goals.

Parameters:

msg() – the result of the action

scripts.service_node_B.goal_rensponse(req)[source]

This function is called when the service is called and returns the number of reached and cancelled goals.

Parameters:

req (GoalRequest) – the request of the service

Returns:

the response of the service

Return type:

GoalResponse

scripts.service_node_B.main()[source]

This function initializes the node, subscribes to the ‘/reaching_goal/result’ topic and provides the service ‘/goal_n’.

Subscriber_node_C Module

This node implements a subscriber node that subscribes to the robots position and velocity (using the custom message) and prints on the console the distance of the robot from the target (set in node action_client_A), and its average speed with the frequency set as a parameter in the launch file.

Subscriber:

/pos_vel

Service client:

/set_frequency

scripts.dist_vel_C.callback(msg)[source]

This callback function subscribes to the ‘/pos_vel’ topic and prints the distance of the robot from the target and its average speed.

Parameters:

msg (Pos_vel) – the position and velocity of the robot

scripts.dist_vel_C.main()[source]

This function initializes the node and subscribes to the ‘/pos_vel’ topic.
